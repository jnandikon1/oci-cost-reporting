# oci-reporting-Hourly

Hourly master playbook to collect all database types, load into a dedicated hourly table, and generate/distribute hourly reports. Variables come from vars/common.yml. Reference scheduling and ADW loading guidance: https://github.com/oracle-samples/usage-reports-to-adw/blob/main/step_by_step_howto.md#7-how-to-schedule-daily-report.
